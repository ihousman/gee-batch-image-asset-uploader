Uploading geoTiffs to Google Earth Engine script module

-Contact: Ian Housman ihousman@fs.fed.us

-Requirements: EE python package, Google Cloud SDK, Python standard install

-Contents: 
	gee_batch_image_asset_uploader_lib.py - Script containing necessary functions for uploading image sets as imageCollection assets in GEE. No user 						parameters found in this script.
	gee_batch_image_asset_uploader.py - Script for user to enter input parameters
	
	image_dir- A set of test images to get started with
